## 什么是 JavaScript ?

**`JavaScript`是一门客户端脚本语言**



特点：

1. 运行在客户端（每个浏览器都有 JS 解析引擎）
2. 脚本语言（不需要编译，直接被浏览器引擎解析）





## 为什么要用 JavaScript ?

在前端三件器中：

`HTML`：显示页面（最基础的页面）

`CSS`：美观页面

`JavaScript`：增强与用户之间的交互，做动态内容

## 1. 了解 NoSQL

### 1.1 什么是 NoSQL?

泛指非关系型数据库

**特点：**

- 易扩展：此类型数据存储不需要固定的模式，无需多余操作就可以横向扩展
- 高性能：非常高的读写性能
- 灵活的数据模型：无需事先为存储的数据建立字段（关系型数据库增删字段会对先前影响较大）

 

### 1.2 如何使用?

- KV
- Cache
- Persistence

## 1. 什么是 lambda 表达式？

`Lambda` 允许把函数作为一个方法的参数（函数作为参数传递进方法中）



## 2. 为什么使用 lambda 表达式？

使用 `Lambda` 表达式可以使代码变的更加简洁紧凑



## 3. 标准格式

- 形式参数 `()`
- 箭头 `->`：用箭头指向后面要做的事情
- 代码块 `{}`：方法体中的内容







## 4. 使用前提

- 有一个接口
- 接口中有且仅有一个抽象方法





## 5. 使用规范

- 参数类型可以省略，但是有多个参数的情况下不能只省略一个
- 如果参数有且仅有一个，`()` 可省略
- 代码块语句只有一条，可以省略 `{}` 和 `;`，甚至 `return`

**内存 (memory)**

Random-access Memory

是与 CPU 直接交换数据的内部存储器，用于存储临时数据。



**硬盘 (hard disk)**

计算机上的非易失性存储器。



**字符**

字符包括英文，数字，符号，语言文字......

不同编码里，字符与字节的关系不同



**字节 (byte)**

### 创建表

```sql
CREATE TABLE mytable(
	# 各个列名用逗号隔开
	num INT,
	name VARCHAR(30),  # VARCHAR(length) 需要指定长度
	age INT，
	score DOUBLE,
	birthday DATE,
	edited_time TIMESTAMP
);
```

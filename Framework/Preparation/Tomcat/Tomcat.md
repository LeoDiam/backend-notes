# Tomcat quickstart

## 什么是服务器

**服务器**

安装了服务器软件的计算机，称为服务器。

**服务器软件**

对用户发出的`request`， 进行处理，再发出`response`的软件

**Web 服务器软件**

用户可通过浏览器（输入网址）来访问你的 web 项目。



## 什么是 Tomcat

Tomcat 就是上述一种服务器软件。
